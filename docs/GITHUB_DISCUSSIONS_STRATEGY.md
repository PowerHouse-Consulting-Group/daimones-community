# 🏛️ NGO Partners - Greek Orthography Feedback

Welcome, NGO partners! This is your dedicated space for providing feedback on Aristotle's Greek responses.

## 📋 How to Provide Feedback

### Option 1: GitHub Issues (Recommended)
For specific errors, create an issue:
- Use the **Aristotle Feedback** template
- Include the query and response
- Provide the correct form
- Cite sources if possible

[Create Aristotle Feedback Issue →](../../issues/new?template=aristotle_feedback.md)

### Option 2: Discussion Thread
For general feedback or questions:
- Post in this thread
- Tag team members with @username
- Share screenshots if helpful

## 🎯 What We Need Help With

### Priority Areas:
1. **Polytonic Accents** - Correct placement of acute, grave, circumflex
2. **Breathings** - Rough (dasia) vs. smooth (psili)
3. **Declensions** - Proper noun/adjective endings
4. **Dialect Forms** - Classical Attic vs. Koine variations

### Example Feedback:
```
Query: "What is virtue?"
Response: "Ἡ ἀρετὴ ἕξις ἐστὶ..."
Issue: Missing rough breathing on ἕξις
Correction: Should be "ἑξις" with dasia
Source: LSJ, Smyth §11
```
