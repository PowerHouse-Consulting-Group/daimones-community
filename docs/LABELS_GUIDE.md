# 🏷️ DAIMONES Issue Labels

We use labels to organize and prioritize issues in our community repository.

## 🎯 Issue Types

| Label | Description | When to Use |
|-------|-------------|-------------|
| `bug` | Something isn't working | Report a bug or error |
| `enhancement` | New feature or request | Suggest an improvement |
| `aristotle` | Aristotle response issues | Greek, translation, or philosophy feedback |
| `documentation` | Documentation improvements | Fix typos, add guides, improve clarity |
| `question` | Further information needed | Ask a question about the platform |

## 🏛️ Aristotle Feedback Categories

| Label | Description |
|-------|-------------|
| `greek` | Polytonic Greek orthography issues |
| `translation` | Translation errors (EN/EL) |
| `philosophy` | Philosophical accuracy issues |
| `source` | Wrong or missing source attribution |

## 📊 Priority Levels

| Label | Description | Response Time |
|-------|-------------|---------------|
| `priority-p0` | Critical - platform broken | < 24 hours |
| `priority-p1` | High priority | < 3 days |
| `priority-p2` | Normal priority | < 1 week |
| `priority-p3` | Low priority / backlog | < 1 month |

## 👥 Community Labels

| Label | Description |
|-------|-------------|
| `good first issue` | Easy issues for newcomers |
| `help wanted` | We need community help |
| `needs-triage` | New issues needing review |
| `confirmed` | Issue has been verified |

## ✅ Status Labels

| Label | Description |
|-------|-------------|
| `in progress` | Someone is working on it |
| `needs-info` | Waiting for more information |
| `resolved` | Issue has been fixed |
| `wontfix` | Will not be addressed |
| `duplicate` | Already reported |

---

## 🎯 How We Prioritize

### P0 - Critical
- Platform down
- Data loss
- Security vulnerabilities

### P1 - High
- Major feature broken
- Blocking university pilots
- Widespread user impact

### P2 - Normal
- Minor bugs
- Feature requests
- Single user impact

### P3 - Low
- Nice to have
- Cosmetic issues
- Future enhancements
